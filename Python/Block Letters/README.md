# Block Letters
ASCII art is a way of creating images using the individual characters from a keyboard.

For example, the following is an ASCII art of some letters from the English alphabet.

![ASCII ART EXAMPLE](https://content.codecademy.com/courses/learn-cpp/hello-world/block-letters.png)

A grid showing eight 5x7 matrices arranged in two rows of four matrices each. Each matrix has column headers numbered 1 through 5 and row headers numbered 1 through 7. The matrices contain letters A through H, respectively, with each letter appearing in different patterns within its matrix. Matrix A shows the letter A in various cells, matrix B shows the letter B in various cells, and so on. Each letter's placement within its respective matrix creates a larger pattern of that letter across the 5 columns and 7 rows. e.g., the A matrix uses the capital character "A" to form a larger picture of capital "A".

The letter F is made up of 5 characters in the first row, one character in the second and third rows, three characters in the fourth row, and one character in the fifth, sixth, and seventh rows. These characters, when viewed together, look like the letter F.

In this project, we will write a Python program that displays the initials of your name in block characters to get an introduction into the world of ASCII art.
