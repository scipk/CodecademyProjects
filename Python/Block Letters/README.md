# Block Letters
[ASCII art](https://en.wikipedia.org/wiki/ASCII_art) is a way of creating images using the individual characters from a keyboard.

For example, the following is an ASCII art of some letters from the English alphabet.

![ASCII ART EXAMPLE](https://content.codecademy.com/courses/learn-cpp/hello-world/block-letters.png)

The letter F is made up of 5 characters in the first row, one character in the second and third rows, three characters in the fourth row, and one character in the fifth, sixth, and seventh rows. These characters, when viewed together, look like the letter F.

In this project, we will write a Python program that displays the initials of your name in block characters to get an introduction into the world of ASCII art.
