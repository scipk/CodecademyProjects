# Restaurant Database
This project provides a hands-on introduction to building a Restaurant Database System using C++. You’ll simulate a real-world application that lets users add, save, and load restaurant records with dynamic menus. Throughout, you’ll strengthen essential C++ stream concepts while tackling meaningful problems.

You will practice:

- Using `ofstream` and `ifstream` for reading and writing structured data
- Formatting output with manipulators such as `fixed` and `setprecision`
- Validating user input with `cin.fail()`, `clear()`, and `ignore()`
- Parsing serialized data using `getline()` and `stringstream`
- Overloading the `<<` and `>>` operators for custom stream behavior
- Managing collections with `vector` and loops
- Creating a simple text-based user interface

By the end, you will have a program that maintains data across runs, similar to a database-backed app.
