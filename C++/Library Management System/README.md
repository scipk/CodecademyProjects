# Library Management System
In this project, you will create a Library Management System in C++ to simulate how a library manages its books, borrows, and reservations. The system will allow users to interact with books, track borrowed titles, and handle the library’s budget.

You will:

- Create a `Library` class to manage the collection of books, allowing users to add new books, borrow books, and list available titles.
- Track the total number of borrowed books across all instances of the library.
- Manage shared variables, allowing multiple files to interact with the same data.
- Implement mutable members to allow for the modification of certain variables even in `const` member functions, like reserving a book while keeping other attributes unchanged.
- Handle variable scope carefully to ensure that data is properly encapsulated, ensuring smooth interactions within the system and access control.

By the end of this project, you’ll have built a working library system that manages books and tracks borrowing statistics while applying key storage class concepts to enhance your code’s efficiency and maintainability.
